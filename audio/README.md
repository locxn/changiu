# Birthday Song Placeholder

Place the background music file in this folder.

## Required filename

The file should be named exactly:

```text
birthday-song.mp3
```

The website expects the song at:

```text
public/audio/birthday-song.mp3
```

## When the song starts

The song starts only after the envelope is clicked. It does not autoplay before user interaction.

## Copyright note

Do not commit copyrighted music publicly unless you have the rights to use and share it.

## Replacing the song later

To replace the song, delete the old `birthday-song.mp3`, add a new mp3 with the same filename, then rebuild and redeploy the website.
